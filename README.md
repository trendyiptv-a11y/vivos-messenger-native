# VIVOS Messenger Native

Native mobile Messenger app for VIVOS, built with Expo, React Native, TypeScript, and Supabase.

## Current scope
- Authentication
- Inbox
- Conversations
- Call history
- Profile
- Push notifications scaffold
- Call session flow scaffold
- Native WebRTC diagnostics scaffold

## Stack
- Expo
- React Native
- TypeScript
- Expo Router
- Supabase
